# MY CSS STUDY

반응형에 대한 이해력을 높이고, 실제 반응형 웹사이트 구현 능력을 키웠습니다.  
서버와의 실시간 통신을 다루고, 실무에서 사용하는 60개 이상의 최신 프로퍼티를 학습하였습니다.  
나만의 일기장을 만들어보며 CSS 기본 원리와 사용법을 학습하였습니다.  
그 후, 나만의 쇼핑몰을 만들어보며 CSS의 플렉스 레이아웃 시스템에 대해 이해하고, 현대 웹 레이아웃 구성에 필수적인 Flex, Grid에 대해 학습하였습니다.  
마무리로 반응형 웹과 transition, keyframes, animation 등의 동적 CSS 요소들에 대해 학습하였습니다.
